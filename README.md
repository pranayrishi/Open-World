# Open World 3D Exploration Game

An open-world 3D exploration game built with Three.js, Rapier physics, and TypeScript.

## Features

- **Procedural Terrain**: Infinite terrain with biomes (plains, forest, desert, tundra, swamp)
- **Water System**: Dynamic water with wave shaders and fresnel effects
- **Vegetation**: GPU-instanced grass, rocks, and trees
- **Characters**: 12 NPCs and 28 animals with animations (idle, walk, run)
- **Day/Night Cycle**: Dynamic sky with sun and stars
- **Physics**: Rapier physics engine for collision and movement
- **Minimap**: Real-time terrain overview

## Controls

- **W/A/S/D**: Move forward/left/backward/right
- **Mouse**: Look around
- **Space**: Jump

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- Three.js - 3D rendering
- Rapier - Physics engine
- TypeScript - Type safety
- Vite - Build tool
